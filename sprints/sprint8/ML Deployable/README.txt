To run the code provided you must first create a Google Colab instance.
This is a free service provided by Google and does not require anything more than an account.

Copy paste the code provided in the 'evaluateGraph' text file into a code cell.
Create a new code cell and copy paste the 'File uploader' code.
Run the 'File Uploader' code and select 'finalized model (6).sav'
Run the 'File Uploader' again and select one of the provided graphs to test.
Copy the file name produced in the output cell into the argument slot of the evaluateGraph method in the last line of the first code block.
Example line: evaluateGraph('guitar_B3_very-long_forte_normal.jpg')

Run the evaluateGraph code block and check your output label!
